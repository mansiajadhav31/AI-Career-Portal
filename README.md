# AI-Career-Portal
AI Career Portal is a full-stack web application that helps job seekers explore career opportunities, AI-powered recommendations, and job management features.
## ✨ Features

- 👤 User Registration & Login
- 💼 Job Search & Job Listings
- 🤖 AI-Based Career Recommendations
- 📄 Resume Upload & Profile Management
- 🔍 Search & Filter Jobs
- 📱 Responsive User Interface
